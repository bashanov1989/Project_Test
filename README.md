# Комплексное тестовое задание

Финальный проект, объединяющий технические навыки аналитика данных.

**Состав проекта:**
- Обработка сырых данных и расчет ключевых метрик эффективности.
- Подготовка аргументированных выводов для бизнеса на основе цифр.
- Визуализация результатов, позволяющая быстро принимать управленческие решения.
